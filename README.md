# 🛰️ TUMULUS
Let's revive Papy's code !!!
A personal project to revive and expand my grandfather’s educational physics simulations — bringing them into the modern Python ecosystem with interactive visualizations and a GUI built using **PySide6 / Qt**.

---

## 📖 About the Project

Years ago, my grandfather developed small programs to visualize fundamental physics concepts such as:

* Satellite orbits around Earth
* Ballistic trajectories
* Real-time force and velocity vectors visualization

These tools were designed for **education**, helping users *see* physics in action rather than just calculate it.

This project aims to:

* 🧬 Recreate those original simulations
* 🎨 Modernize the interface and visuals
* 🚀 Extend the simulations with new features
* 📚 Serve as a personnal learning journey into **Qt programming with PySide6**

---

## ✨ Features (Planned & In Progress)

* 🌍 Orbital mechanics simulation (Earth–satellite systems)
* 🎯 Projectile motion and ballistic trajectories
* 🧭 Real-time vector visualization (forces, velocity, acceleration)
* ⏯️ Interactive controls (pause, reset, parameter tuning)
* 📊 Optional graphs (position, velocity, energy over time)
* 🖥️ Desktop GUI built with PySide6

---

## 🛠️ Tech Stack

* **Python 3**
* **PySide6 (Qt for Python)** – GUI framework
* **NumPy** – numerical computations
* **Matplotlib / PyQtGraph (TBD)** – plotting and visualization

---

## 🎓 Educational Goals

This project is not just about rebuilding old simulations — it's also a way to:

* Learn **Qt / GUI programming**
* Make basic physics **accessible to everyone**
* Explore **real-time rendering and interaction**
* Practice writing clean, modular, extensible code

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/peilclem/TUMULUS.git
cd TUMULUS
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the application

```bash
python main.py
```
